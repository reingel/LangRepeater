
# LangRepeater

## 개요
- 외국어 학습을 위하여 mp3 또는 mp4 파일과 srt 파일을 불러와서 srt 파일에 저장된 timestamp 만큼의 segment를 반복적으로 들려주는 프로그램
- 터미널에서 실행하고 키보드 입력만으로 실행되는 백엔드 위주의 프로그램

## 개발언어
- 맥용 터미널에서 실행되는 프로그램을 개발할 수 있는 언어를 사용하되 향후 아이폰, 아이패드, 안드로이드, 웹 앱으로 확장할 수 있는 가능성을 고려한다.

## 기능 요구사항
- 터미널에서 LangRepeater(이하 LR)를 입력하면 실행되고 간단한 안내문구를 출력한다.
- LR은 동일 폴더에서
    - mp3, mp4 파일를 검색하고
        - 있으면 목록을 보여주고 선택하라고 물어본다.
        - 선택된 파일을 미디어 파일로 사용한다.
    - 다음으로 srt 파일을 검색하고
        - 있으면 목록을 보여주고 선택하라고 물어본다.
        - 선택된 파일을 자막 파일로 사용한다.
    - 없으면 경로를 입력하라고 한 후 그 경로에서 검색한 후 위 작업을 수행한다.
- 미디어 파일과 자막 파일이 선택되면 첫 번째 자막 3개를 보여준다. 이 때 첫 번째 자막을 볼드체로 표시한다.
- 사용자가 아래 '키보드 매핑'에서 지정한 재생키를 누르면 첫 번째 자막에 해당하는 음성 구간을 재생한다. (mp4의 경우 영상이 아닌 음성만 재생)
- 재생키를 다시 누르면 방금 재생한 구간을 다시 재생한다.
- 다음 구간 키를 누르면 다음 자막 구간을 재생하고 이전 자막, 현재 자막, 다음 자막을 보여준다. 이 때 현재 자막을 볼드체로 표시한다.
- 이전 구간 키를 누르면 이전 자막 구간을 재생하고 마찬가지로 3개의 자막을 표시한다.
- 학습에 대한 통계를 기록하기 위하여 진행률(현재 자막 순서 / 전체 자막 수, 단위 %), 각 자막별 재생 횟수, 총 재생 횟수 등을 stat.yaml 파일의 해당 미디어 파일명에 해당하는 섹션에 저장한다.
- 실행종료 키를 누르면 프로그램을 종료한다.
- 실행종료하기 전에 현재 사용 중인 미디어 파일, 자막 파일, 재생 구간 정보를 progress.yaml 파일에 저장한다.
- 다음 번 실행시 progress.yaml 파일이 있으면 읽어와서 정보를 출력하고 이어서 할 것인지 새로운 파일을 열 것인지 선택하게 한다.
- 새로운 파일을 선택하여 학습을 진행하다가 실행종료하면 progress.yaml에 이전 기록을 지우지 말고 현재 학습 정보를 추가로 기록한다.
- 다음 번 실행 시 progress.yaml에 두 개 이상의 학습정보가 있을 경우 먼저 어느 학습을 진행할 것인지 물어본다.

## 키보드 매핑
- 재생: 스페이스, S
- 다음 구간: 오른쪽 화살표, D
- 이전 구간: 왼쪽 화살표, A
- 실행종료: Q, ESC
